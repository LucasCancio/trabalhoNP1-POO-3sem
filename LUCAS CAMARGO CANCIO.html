Segue instruções para realização do seu trabalho personalizado da disciplina Linguagem de Programação Orientada a Objetos.

Informo que as instruções devem ser seguidas rigorosamente, e que o código entregue deve compilar para que o trabalho seja considerado.

Uma dica é compilar o código a cada item concluído, e só partir para o próximo item após a correção dos problemas do item atual.

É necessário aplicar a correta convenção de nomes para os identificadores, conforme exemplos:

  • Classe, ClasseComNomeComposto
  • atributo, atributoComNomeComposto
  • metodo, metodoComNomeComposto
  • CONSTANTE, CONSTANTE_COM_NOME_COMPOSTO

Implemente na sequência definida para facilitar a conclusão do trabalho

  1. Criar a classe ContaBancaria com os seguintes atributos: numero do tipo String e saldo do tipo double.
  2. Encapsular fortemente os atributos, ou seja, tornar numero, saldo privados e implementar os métodos de acesso (get e set).
  3. Criar uma nova classe AgenciaBancaria com o seguinte atributo encapsulado: codigo do tipo int.
  4. Criar um construtor na classe AgenciaBancaria que receba e inicialize o seguinte atributo: codigo.
  5. Na AgenciaBancaria, criar um método público para imprimir, na saída padrão (System.out.println()), todos os atributos da classe AgenciaBancaria com a seguinte assinatura: imprime() além do tipo de retorno void.
  6. Voltar a classe ContaBancaria e incluir um atributo agenciaBancaria do tipo AgenciaBancaria e o encapsular fortemente.
  7. Criar um método público para imprimir, na saída padrão (System.out.println()), todos os atributos da classe ContaBancaria com a seguinte assinatura: imprime() além do tipo de retorno void.
  8. Ainda na classe ContaBancaria, criar um construtor que receba os três atributos (numero, saldo, agenciaBancaria) e os e inicialize.
  9. Realize uma sobrecarga (overload) do construtor da ContaBancaria, que receba os dois atributos (numero, saldo) e os inicialize. Lembre que o construtor pode chamar um construtor da própria classe (this(lista de parâmetros correta), altere o construtor definido no item anterior).
  10. Criar uma nova classe ContaPoupanca que estenda (extends) a ContaBancaria com o seguinte atributo: diaAniversario o tipo short e o encapsular fortemente.
  11. Criar na classe ContaPoupanca, um construtor que receba os atributos de instânica da classe ContaBancaria (numero, saldo, agenciaBancaria) e o atributo diaAniversario e faça a inicialização de todos atributos. Lembre que pelo menos um construtor da classe deve chamar o construtor da super classe (super(lista de parâmetros correta)).
  12. Na classe ContaPoupanca, faça a sobreposição/sobreescrita (override) do método imprime() definido na classe ContaBancaria, para que além dos atributos definidos em ContaBancaria, também seja impresso o atributo definido em ContaPoupanca.
  13. Finalizando crie uma classe Teste que deverá ser executada, ou seja, possuir o método main (public static void main(String[] args)). No método main deve ser realizado o seguinte código:
